# Refund Site

A Node.js + Express web application that connects to the GiantBomb API to search for games and display basic information.

## Features
- Backend built with **Express** and **Node.js**
- Uses the **GiantBomb API** for real-time game data
- Environment variables handled with **dotenv** for API key security
- Clean, minimal structure for easy modification and expansion
